# Gabb Pinball

Web-based pinball arcade game branded for [Gabb Wireless](https://gabb.com). Built with Phaser 3 + Vite + TypeScript.

> Kids will be kids. Get a pinball machine that lets them.

**▶ Play:** https://gabb-pinball.vercel.app
**Source:** https://github.com/aaronmatson1/gabb-pinball

## Play

- **Z / Left Arrow** — left flipper
- **M / Right Arrow** — right flipper
- **Space** — hold to charge plunger, release to launch
- **Tap left/right** on touch screens

Hit the four product targets (Watch, Phone, Music, Messenger) to complete a mode and score a 5,000-point bonus.

## Run locally

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # serve the build
```

## Deploy

Deployed automatically to Vercel from the `main` branch.

## Brand

Colors and theming are inspired by Gabb Wireless's kid-safe, family-first identity. Not an official Gabb product — built as a portfolio piece.
